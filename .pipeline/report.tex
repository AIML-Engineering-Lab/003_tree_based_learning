\section{Project Overview}

I tackle EV battery thermal runaway and wafer yield prediction using Decision Trees and Random Forests. By leveraging ensemble learning, I enhance predictive performance significantly. The models are carefully tuned and evaluated, achieving AUC values up to 1.0000 for the battery dataset and robust results for wafer predictions.

\textbf{Decision Tree:} Creates interpretable IF-THEN rules through data splitting.

\textbf{Random Forest:} Combines multiple trees to reduce variance and improve accuracy.

\textbf{Feature Importance:} Identifies which features best predict outcomes across trees.

\textbf{Class Imbalance:} Addresses skewed classes using balanced weights during training.

\textbf{Ensemble Learning:} Multiple weak learners outperform a single strong learner through averaging.
